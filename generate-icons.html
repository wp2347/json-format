JSON Formatter - Icon Generator

📋 Instructions

  1. Click "Generate Icons" to create all required icon sizes
  2. Right-click on each icon and select "Save image as..."
  3. Save them to your dist/icons/ folder with the correct names:
    • 16x16 → icon-16.png
    • 32x32 → icon-32.png
    • 48x48 → icon-48.png
    • 128x128 → icon-128.png
    • 512x512 → icon-512.png
